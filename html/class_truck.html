void Truck::readTrucks ( const std::string & filename)

Loads the truck’s capacity and pallet count from a CSV file.

Expects the file to contain two comma-separated values: capacity,pallets on a single line (after an optional header). On failure to open the file, writes an error to std::cerr.

Parameters
