Reads the truck and pallet files corresponding to a dataset index. 

Reads truck and pallet information from two CSV files.

Constructs file paths\+: "{}../datasets/\+Truck\+And\+Pallets\+\_\+0$<$files\+Number$>$.\+csv"{} "{}../datasets/\+Pallets\+\_\+0$<$files\+Number$>$.\+csv"{} Opens each file in turn, parses the header and data lines, and fills\+:
\begin{DoxyItemize}
\item truck.\+capacity
\item truck.\+pallets (number of pallets expected)
\item pallets (vector of \doxylink{class_pallet}{Pallet} objects)
\end{DoxyItemize}

If a file fails to open, an error is printed and the function returns early.

Given a string suffix {\ttfamily files\+Number}, this function constructs the paths\+: "{}../datasets/\+Truck\+And\+Pallets\+\_\+0$<$files\+Number$>$.\+csv"{} "{}../datasets/\+Pallets\+\_\+0$<$files\+Number$>$.\+csv"{} It first opens the truck file and reads the header plus one data row to set {\ttfamily truck.\+capacity} and {\ttfamily truck.\+pallets}. Then it opens the pallets file, skips its header, and reads up to {\ttfamily truck.\+pallets} entries, converting each line into a \doxylink{class_pallet}{Pallet} object and appending it to {\ttfamily pallets}.

If either file fails to open, an error message is written to std\+::cerr and the function returns early without throwing exceptions.
